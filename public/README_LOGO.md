# Sophos Logo Setup

To complete the branding setup, please download the Sophos logo files from [https://brand.sophos.com/](https://brand.sophos.com/) and place them in this `public` directory.

## Required Files:

1. **sophos-favicon.png** (or .ico)
   - Size: 32x32 or 16x16 pixels
   - Format: PNG or ICO
   - Used for: Browser tab favicon

2. **sophos-logo-white.svg** (or .png)
   - Format: SVG (preferred) or PNG with transparent background
   - Color: White/light version for use on blue background (#005BC8)
   - Used for: Header logo

## Steps:

1. Visit https://brand.sophos.com/
2. Download the Sophos logo files (favicon and white logo)
3. Place the files in this `public` directory with the exact filenames:
   - `sophos-favicon.png`
   - `sophos-logo-white.svg`

The application will automatically use these files once they are placed in the public directory.

## Fallback:

If the logo files are not found, the application will fall back to using a Shield icon.
